# History Detail Action-First Sheet

Date: 2026-06-21

## Goal

Make the History detail sheet useful immediately after opening it.

## Change

- Reordered the detail sheet to show actions first.
- Kept the preview/player card directly after actions.
- Moved project metadata into a collapsed `Project details` disclosure.
- Moved event history into a collapsed `Activity` disclosure.
- Reordered actions so saved-reel preview and share appear before watching the source video.

## Product rationale

Users open a saved project to resume, preview, save, or share. Metadata and event logs are still useful for launch/debugging, but they should not block the core task.
