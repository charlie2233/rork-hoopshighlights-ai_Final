# History Rename Affordance

## Goal

Make project rename easier to discover without adding another History section or more explanatory copy.

## Change

- Replaced the tiny title pencil-only icon with a compact inline `Rename` chip.
- Kept the same tap target and rename behavior.
- Did not add another card, row, badge group, or settings control.

## User impact

A normal user can tell project names are editable without guessing that the title or tiny pencil is tappable.

## Product note

This supports the cleaner friendly-name work while keeping History action-first and not stacked.

## Validation

Not run in this pass per instruction to avoid extra simulator/build/test work unless explicitly requested.
