# Review Priority Card Declutter

## Goal

Make Review feel less word-heavy while keeping the priority-review shortcut available.

## Change

- Removed the extra explanatory summary sentence from the `Review these first` priority card.
- Kept the priority shortcut, icon, and clip count.
- Full clip evidence and review details remain available in the focused clip/detail views.

## User impact

A normal user sees a simple shortcut instead of another paragraph above the clip decision flow.

## Product note

This keeps Review focused on the current decision: watch the clip, Keep or Nah, move on.

## Validation

Not run in this pass per instruction to avoid extra simulator/build/test work unless explicitly requested.
